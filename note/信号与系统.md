# 信号与系统

# 第一章 绪论

## 1.1 信号的概念

广义地说, 信号是随着时间变化的某种物理量.

信号可分为以下几类：

#### a. 按确定性分类
- **确定信号**：给定时间值，可确定唯一函数值的信号，是理想的、可精确预测的模型。
- **随机信号**：给定时间值，函数值仅能用概率描述的信号，工程中绝大多数实际信号（含噪声、干扰）均为此类，需用统计方法分析。

---

#### b. 按时间变量特性分类
- **连续时间信号（模拟信号）**：时间变量为连续值，允许存在有限个不连续点，如语音、电路中的电流电压。
- **离散时间信号**：时间变量仅取离散时刻值，如水文观测的定时记录水位。
- **数字信号**：时间和幅值均取离散值的信号，适合计算机存储与处理，抗干扰能力强。

---

#### c. 按周期性分类
- **周期信号**：满足 \( f(t)=f(t+T) \)（\( T \) 为周期），以固定周期无始无终重复，如正弦/余弦信号；工程中常将长时间近似重复的信号按周期信号分析。
- **非周期信号**：不满足周期性条件的信号。

---

#### d . 按能量/功率特性分类
以1Ω负载上的能量与功率定义为标准：
- **能量信号**：总能量为有限非零值，平均功率为0，如有限时长脉冲信号。
- **功率信号**：平均功率为有限非零值，总能量为无穷大，如周期信号。

信号 $f(t)$ 延时 $t_0$ 后的信号表示为 $f(t-t_0)$，其中 $t_0>0$ 表示信号被延迟了 $t_0$ 时间单位。

#### c.信号的尺度变换与反褶

信号 $f(t)$ 进行尺度变换后得到的信号表示为 $f(at)$，其中 $a$ 是一常数。当 $a>1$ 时，信号被压缩；当 $0<a<1$ 时，信号被拉伸, 当 $a=-1$ 时，信号被反褶。

## 1.3 系统的概念

从一般的意义上说，系统是一个由若干互有关联的单元组成的、具有某种功能、用来达到某些特定目标的有机整体。

不同的系统具有各种不同的特性。按照系统的特性，系统可做如下的分类。

#### a. 按系统的线性与非线性分类
- **线性系统**：满足叠加原理的系统，即输入的线性组合对应输出的线性组合。
- **非线性系统**：不满足叠加原理的系统，输出与输入之间存在非线性关系。

#### b. 按系统的时变与时不变分类
- **时不变系统**：系统的特性不随时间变化，即系统的输入输出关系在任何时间都是相同的。
- **时变系统**：系统的特性随时间变化，即系统的输入输出关系在不同时间可能不同。

#### c. 按系统的时间连续与时间离散分类
- **连续时间系统**：系统的输入输出信号为连续时间信号。
- **离散时间系统**：系统的输入输出信号为离散时间信号。

#### d. 按系统的因果与非因果分类
- **因果系统**：系统的输出仅依赖于当前和过去的输入信号。
- **非因果系统**：系统的输出依赖于未来的输入信号。

## 1.5 微分方程求解

### 核心通用步骤

无论齐次还是非齐次，第一步永远是一样的
1.  **写出特征方程**：对于 $y'' + py' + qy = f(x)$，其特征方程为 $\lambda^2 + p\lambda + q = 0$。
2.  **求特征根** $\lambda_1, \lambda_2$。
3.  **写出齐次通解** $y_h$。

---

###  二阶常系数**齐次**线性微分方程
**形式**：$y'' + py' + qy = 0$

**通解结构**：$y = y_h$

根据判别式 $\Delta = p^2 - 4q$ 或特征根的情况，分三种情形

| 特征根情况 | 条件 | 齐次通解 $y_h$ |
| :--- | :--- | :--- |
| **两个不相等实根** | $\lambda_1 \neq \lambda_2$ | $y = C_1 e^{\lambda_1 x} + C_2 e^{\lambda_2 x}$ |
| **两个相等实根** | $\lambda_1 = \lambda_2 = \lambda$ | $y = (C_1 + C_2 x) e^{\lambda x}$ |
| **一对共轭复根** | $\lambda_{1,2} = \alpha \pm i\beta$ | $y = e^{\alpha x} (C_1 \cos \beta x + C_2 \sin \beta x)$ |

> **注意**：$\alpha = -\frac{p}{2}, \beta = \frac{\sqrt{4q-p^2}}{2}$

---

### 二阶常系数**非齐次**线性微分方程
**形式**：$y'' + py' + qy = P_m(x)e^{\gamma x}$
*   $P_m(x)$：$m$ 次多项式
*   $\gamma$：常数

**通解结构**：$y = y_h + y_p$
*   $y_h$：同上，先求齐次通解
*   $y_p$：**特解**，使用**待定系数法**求解

#### 特解 $y_p$ 的设法

设特解形式为：
$$ y_p = x^k Q_m(x) e^{\gamma x} $$

这里有两个关键点需要确定

#### A. 确定多项式 $Q_m(x)$
$Q_m(x)$ 是一个**通用的** $m$ 次多项式，系数待定
*   若 $m=0$（常数），设 $Q_m(x) = A$
*   若 $m=1$（一次），设 $Q_m(x) = Ax + B$
*   若 $m=2$（二次），设 $Q_m(x) = Ax^2 + Bx + C$
*   ...以此类推

#### B. 确定修正因子 $k$
$k$ 取决于 $\gamma$ 与特征根 $\lambda$ 的关系

| $\gamma$ 与特征根的关系 | $k$ 的值 | 说明 |
| :--- | :---: | :--- |
| $\gamma$ **不是**特征根 | $0$ | 直接设 $Q_m(x)e^{\gamma x}$ |
| $\gamma$ 是**单**特征根 | $1$ | 需乘 $x$，即 $x Q_m(x)e^{\gamma x}$ |
| $\gamma$ 是**重**特征根 | $2$ | 需乘 $x^2$，即 $x^2 Q_m(x)e^{\gamma x}$ |

> **记忆技巧**：$k$ = $\gamma$ 在特征根中出现的次数

#### C. 求解步骤
1.  根据 $k$ 和 $m$ 写出 $y_p$ 的表达式。
2.  求出一阶导 $y_p'$ 和二阶导 $y_p''$。
3.  代入原方程 $y'' + py' + qy = P_m(x)e^{\gamma x}$。
4.  消去两边的 $e^{\gamma x}$（因为它恒不为0）。
5.  比较两边 $x$ 的同次幂系数，列出方程组。
6.  解出 $Q_m(x)$ 中的待定系数。
7.  最终通解 $y = y_h + y_p$
